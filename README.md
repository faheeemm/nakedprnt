# nakedprnt

A minimal Java printing utility built on a simple idea:

Make output cleaner.

`nakedprnt` provides a lightweight and extensible foundation for console printing while keeping the API straightforward and easy to use.

## Features

- Minimal and focused
- Lightweight
- Dependency-free
- Extensible by design
- Clean developer experience

## Philosophy

Software often grows from small abstractions.

`nakedprnt` starts with a simple printing interface and leaves room for experimentation, customization, and future enhancements without sacrificing simplicity.

## Status

Early-stage project. Small by design.
